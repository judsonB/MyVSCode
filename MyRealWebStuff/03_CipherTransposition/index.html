Transposition Cipher

The Transposition Cipher is a permutation cipher. Rather than changing letter values you change letter locations. You start with a message and a key word. The keyword is written, then the message is written below it, wrapping the text once the message hits the end of the key word. If there are extra spaces under the keyword unfilled, they can be filled with random letters. Each column is then alphabitzed based on the key word letters (LION becomes ILNO), then put the words back into a String in the order of the columns.
